Квадратные уравнения

Теория

Квадратное уравнение - это уравнение вида: \(ax^2 + bx + c = 0\), где коэффициенты a, b и c — это произвольные числа, причем \(a \ne 0\), а коэффициенты b и c - любые числа, в том числе и 0. То есть, квадратным называется уравнение, где есть \(x^2\).

Квадратное уравнение может:

  • Не иметь корней;
  • Иметь один корень;
  • Иметь два различных корня.

Это зависит от дискриминанта.

Формула нахождения:

\(D = b^2 - 4ac\), где a,b,c - коэффициенты квадратного уравнения.

Вот, например, для уравнения:

\(x^2 - x - 6 = 0\)

\(D = (-1)^2 - 4 * 1 * (-6) = 1 + 24 = 25\)

Если:

  • \(D < 0\), корней нет;
  • \(D = 0\), будет ровно один корень;
  • \(D > 0\), корней будет два;

Корни квадратного уравнения

Теперь перейдем к решению уравнения.

  • Если дискриминант D > 0, корни можно найти по формулам: \(x_1 = {-b + \sqrt D \over 2a} \) \(x_2 = {-b - \sqrt D \over 2a} \)
  • Когда D = 0, можно использовать любую из этих формул — получится одно и то же число, которое и будет ответом, потому что если прибавлять или отнимать 0, то ничего не будет изменяться.
  • И, если D < 0, то уравнение не имеет решений.

Совет: перед тем как начать решать квадратное уравнение лучше всего сделать коэффициент перед \(x^2\) равный 1 или хотя бы сделать его максимально минимальным и положительным, так уравнение будет решать намного легче.

Пример 1:

\(x^2 + 3x - 4 = 0\)

Решение:

\(x^2 + 3x - 4 = 0\)

\(D = 9 - 4 * (-4) = 9 + 16 = 25 = 5^2\)

\(x_1 = {(-3 + 5)\over 2} = {2 \over 2} = 1\)

\(x_2 = {(-3 - 5)\over 2} = {-8 \over 2} = -4\)

Неполные квадратные уравнения

Существуют также неполные квадратные уравнения. Это когда коэффициент b или c или равняется нулю, но не a, если коэффициент a = 0, то уравнение будет уже не квадратным. Неполные квадратные уравнения можно решать тоже через дискриминант или другими способами.

Пример 2:

\(x^2 - 1 = 0\)

Решение:

\(x^2 - 1 = 0\)

\(D = 0^2 - 1 * (-4) = 4 = 2^2\)

\(x_1 = {(0 + 2)\over 2} = 1\)

\(x_2 = {(0 - 2)\over 2} = -1\)

Пример 3:

\(3x - 6x^2 = 0\)

Решение:

\(3x - 6x^2 = 0\) | \( * -{1\over3}\)

\(-x + 2x^2 = 0\)

\(2x^2 - x= 0\)

\(D = 1^2 - 4 * 2 * 0 = 1\)

\(x_1 = {(1 + 1)\over 4} = 0.5\)

\(x_2 = {(1 - 1)\over 4} = 0\)
